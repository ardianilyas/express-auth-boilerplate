# Express TS Auth Boilerplate
This repository provide auth boilerplate using express ts, so you no need wasting time to setup auth.

## Tech Stack :
- ExpressJS
- TypeScript
- Prisma ORM (v7)
- Zod (v5)
- JWT (Auth strategy)
- Bcryptjs (Password hashing)
- Pino (Logging)

This folder using Feature Based and Service Repository Pattern, with error handling and auth middlewares.
